🔍 Focus on: parseInt(bin, 2)

➤ What is parseInt()?

parseInt() is a built-in JavaScript function that converts a string into an integer.

➤ What does the second argument 2 mean?

The second argument is the radix (or base) of the number system:

  • 2 means you're interpreting the input as a binary number.

  • So, parseInt("101", 2) means: “Convert the binary number 101 to decimal”.

💡 Example:

javascript
parseInt("101", 2)  // returns 5

Because:

  • Binary 101 = 1×2² + 0×2¹ + 1×2⁰ = 4 + 0 + 1 = 5

🔁 Then why .toString(10)?

After getting the number in decimal, .toString(10) converts it back to a string in base 10.
It’s optional here since parseInt() already gives a number, and innerHTML can handle numbers.

You could write this more simply:

javascript
return parseInt(bin, 2);

But if you always want to be sure it's a base-10 string, keeping .toString(10) is fine.

✅ Summary:

  • parseInt(bin, 2) → Converts binary to decimal number.

  • .toString(10) → Converts that number to a base-10 string.

  • Together: Converts binary input into a decimal string.
