Width vs. Flex-Basis: Seeing the Difference

Scenario 1: Using `width` with `flex-shrink: 0`

Container width: 300px. Items have `flex-grow: 0` and `flex-shrink: 0`.

Desired total width: 100px (A) + 100px (B) + 150px (C) + 30px (margins) = 380px

Since `flex-shrink: 0` prevents shrinking, **item C will overflow** the container.
